# PIGPULL #

## Síntese

Dispositivo eletrônico embarcado PigPull, autoria GreenTech.

## Materiais
### Sistema Elétrico
- 1 ESP32
- 1 Resistor 330 ohm
- 1 Mini DFPlayer
- 1 Auto Falante 0,5W
- 1 Cartão de Memória SD
- 4 Potenciômetros Deslizantes 10k

### Sistema Mecânico
- 4 Guias Lineares 120mm
- 4 Rolamento Linear

### Sistema de Blindagem
- Caixa de Passagem Blindada 150x100x70mm
